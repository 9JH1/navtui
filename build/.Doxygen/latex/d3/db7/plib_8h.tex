\doxysubsection{Detailed Description}
Basic argument parsing library. 

Plib gives functions to create and manage the system arguments of any given function, these arguments can be defined with many options for example shorthand and long-\/hand versions of the same ag eg {\ttfamily -\/-\/help} and {\ttfamily -\/h}.

Arguments are dynamically allocated and free\textquotesingle{}d automatically on exit so you dont have to worry about any memory leaks. Along with this plib provides a builtin function called pl\+\_\+help which prints out all of the arguments in a standard format.

In order to get started with plib first you must include the system arguments into your main function, for reference look here\+: \href{https://www.ibm.com/docs/en/zos/2.4.0?topic=functions-main-function}{\texttt{https\+://www.\+ibm.\+com/docs/en/zos/2.\+4.\+0?topic=functions-\/main-\/function}}

To create an argument use pl\+\_\+a PL\+\_\+A or PL\+\_\+P, read the structure declaration \doxylink{structpl__arg}{pl\+\_\+arg} to learn all of the options.
