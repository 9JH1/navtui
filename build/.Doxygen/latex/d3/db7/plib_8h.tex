node structure 

This is the structure that recursivly holds all of the arguments information, each node can have another node nested in it. This approach seems strange seeing as you could just use an array right? WRONG!! if you try to use an array and lets say this array is dynamically allocated and has a starting ~\newline
 capacity of 2, if for example the pl\+\_\+a function returned the arguments memory pointer and then the array was reallocated then the address of the pointer is removed and therefore causes a dangling pointer, using nodes means that you do not need to reallocate to add a new item as you just need to allocate the space in the node with the highest recursive level.
